# Quick Start Guide

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Key Features to Try

1. **Browse Pizzas**: View the menu on the home page
2. **Filter & Search**: Use the filter button to filter by category, price, or ingredients
3. **Add to Cart**: Click on any pizza card and adjust quantity, then click "Add"
4. **View Details**: Click on a pizza name or image to see detailed information
5. **Order Summary**: Check the right sidebar to see your current order
6. **Get Discounts**: Add 3 or more of the same pizza to get a 10% discount!
7. **Add Custom Pizza**: Navigate to "Add Pizza" to create your own pizza
8. **View Charts**: See data visualizations of prices and order distributions
9. **Confirm Order**: Click "Confirm Order" to save your order (stored in localStorage)

## Testing

Run tests with:
```bash
npm test
```

## Build for Production

```bash
npm run build
npm start
```
